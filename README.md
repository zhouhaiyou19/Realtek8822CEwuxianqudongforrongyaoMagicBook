1. **下载驱动文件**

   2. **解压文件**
      - 将下载的压缩包解压到您选择的目录。

      3. **安装驱动**
         - 打开设备管理器。
            - 找到并右键点击未识别的无线网卡设备，选择“更新驱动程序”。
               - 选择“浏览我的计算机以查找驱动程序软件”。
                  - 选择解压后的驱动文件夹，然后点击“下一步”进行安装。
